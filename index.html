  1. História da Web

    2. (inicio)

    Os primeiros computadores foram criados para ser uma calculadora e foram responsáveis diretamente pela evolução dos computadores como conhecemos.

    • 1ºgeração- Iniciou em 1944 com o computador Harvard Mark I americano uma máquina grande de cálculo eletromecânico, depois em 1946 criam o computador Colossus alemão que era responsável por fazer cálculos e criptografias e no mesmo ano o Estados Unidos lançou o ENIAC que era responsável em calcular as rotas dos mísseis.

    • 2ºgeraçã- 1959 a 1965 marcada pela evolução da troca das válvulas pelos transistores.

    • 3ºgeração- 1965 a 1970 marcada pelo uso dos circuitos integrados, transistor capacitores e resistores.

    • 4ºgeração- 1971 até hoje com uso de microprocessadores e chips cada vez evoluindo mais permitindo criar não somente os computadores pessoais os PCs, mas os dispositivos portáteis.

    Como surgiu

    O primeiro sistema de comunicação iniciou com o telegrafo. Em 1958 um cabo foi traçado através dos oceanos conectando todos os continentes, assim dando partida a comunicação intercontinental.
    Durante a Guerra Fria em 1950, Joseph Carl Robnet Licklider trabalhou na Darpa (Defense Advenced Research Projects Agency) nos Estados Unidos e foi lá que ele criou a ideia inicial do que seria a Arpanet.
    A Arpanet, foi o primeiro sistema de comunicação de pacotes de dados, incluindo "checkpoints". A primeira conexão entre o Stanford Research Institute (SRI) e a University of Carlifornia (UCLA) ocorreu em 29 outubro de 1969. A transmissão era a palavra login, mas chegou apenas a palavra "lo".

  2. Client X Server

    3. (inicio)

    Clients são pessoas que interagem e consumem a informação pela internet como meio de comunicação. essas informações são dos servidores que hospedam os sites que são acessados. Muitas dessas informações são consumidas por meio dos navegadores ou Browsers sendo aplicativos que são instalados nos dispositivos, que interpretam os arquivos enviados pelos servidores. A grande maioria dos navegadores são gratuitos, também seguem padrões estabelecidos pela W3C.

    Os Servers podem ser dos tipos arquivo, segurança (Firewall), Streaming, E-mail e Web. onde uma rede de computadores conectados tem acesso a arquivos e informações ou servidores de internet, onde estão hospedados HTML, CSS, JS, imagens e áudio, dessa forma o navegador recodifica os arquivos e as informações que serão exibidas na tela do dispositivo dos Clients.

    Aplicações Web

    Aplicativos web são produtos desenvolvidos que usam a internet como meio de comunicação entre Client X Serve, sem necessidade de instalação prévia no dispositivo do cliente. Apesar de semelhanças, as aplicações web e os aplicativos possuem diferentes caminhos profissionais para os desenvolvedores.

  3. O que é linguagem de programação?

    4. (inicio)

    "A linguagem de programação é um linguagem escrita e formal que especifica um conjunto de instruções e regras usada para gerar programas (softwares)."
    HTML não é programação. Muitas pessoas pensam que o HTML é uma linguagem de programação, mas na verdade é uma linguagem de marcação de hipertexto que usa tags para definir o comportamento. O HTML é fundamental para o desenvolvimento web porque é a base de todo o desenvolvimento.

    Server-Side - dentro dos servidores existe diferentes linguagens que são executadas somente em servidores, as linguagens de programação mais usadas para software são PHP, Ruby, C#, JS e Java por meio da contribuição do NodeJS.

    Client-Side - o JavaScript é a linguagem mais utilizada no campo do cliente. É essencial que o JS utilizado no lado do cliente seja apoiado pelo módulo V8.
